# AI-Based WhatsApp Message Translator with Voice Assistant

# Overview
This project is an AI-powered tool that translates WhatsApp messages in real-time and converts them into speech to improve accessibility.

# Features
- Real-time message translation
- Multi-language support (Hindi, Marathi, Tamil, Telugu)
- Voice output using AI Text-to-Speech
- Simple click-based interaction

# Technology Used
- Python
- Selenium
- Deep Translator
- Edge TTS

# Objective
To help non-English and illiterate users understand WhatsApp messages easily.

# How to Run
1. Install dependencies:
   pip install selenium webdriver-manager deep-translator edge-tts

2. Run:
   python main.py
